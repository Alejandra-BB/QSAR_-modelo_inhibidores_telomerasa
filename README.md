# 🧬 QSAR para Inhibidores Covalentes de Telomerasa

[![Python 3.8+](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![RDKit](https://img.shields.io/badge/RDKit-2023-green.svg)](https://www.rdkit.org/)

Modelo QSAR con redes neuronales profundas para predecir inhibidores covalentes de la enzima telomerasa.  
**Estado actual:** Construcción de la base de datos de inhibidores.

## 🎯 Objetivo del proyecto

- Recopilar inhibidores conocidos de telomerasa desde ChEMBL y PubChem
- Calcular descriptores moleculares (RDKit, PaDEL)
- Implementar modelos de IA (DNN, XGBoost, Random Forest)
- Predecir nuevas moléculas con potencial inhibitorio

## 📍 Estado actual del proyecto

✅ En progreso: **Construcción de la base de datos**  
⏳ Siguiente: Cálculo de descriptores moleculares  
⏳ Futuro: Entrenamiento de modelos QSAR
